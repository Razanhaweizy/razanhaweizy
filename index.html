Sharing Information

This project is a messaging system that stores messages in text files with metadata for efficient management. I implemented features for handling storage, automatic deletion of outdated messages, and merging files when needed to optimize performance. The system ensures that messages are organized and accessible while maintaining scalability.

GraphQuest

This project explores graph data structures and their practical applications. I implemented a new graph representation, EdgeArrayGraph, that adheres to a generic interface. Additionally, I applied breadth-first search (BFS) to find the shortest paths between nodes and planned for using graphs to manage schedule conflicts. The project emphasizes both conceptual understanding and effective testing strategies.

Pandas

This project focused on maintaining and analyzing patient visit data at medical clinics, utilizing pandas for data manipulation. I worked on cleaning and normalizing patient information from CSV files, updating patient and visit records, and producing data visualizations to track key health metrics. Additionally, I wrote tests to ensure data accuracy and the proper functionality of data processing functions.

Totally Tubular Tables

For this project, I analyzed the density of grocery and convenience stores in various counties across the USA. I worked with multiple datasets to calculate statistics such as the variability of stores across counties and the relationship between population size and store density. I also implemented a summary-generator function that takes in data and computes flexible statistics like the mean or total for different store categories, helping to answer key analysis questions with the datasets provided.

Frog Maze Craze

In this partner project, I developed a maze game where the player controls a frog navigating through a maze of water and grass tiles. The bee collects widgets that affect stamina or portals that allow teleportation, while avoiding running out of stamina or getting stuck. I implemented the game logic using recursion, datatypes, and a reactor model, and used spreadsheets to configure the maze layout and item placements, creating an interactive and engaging gameplay experience.
